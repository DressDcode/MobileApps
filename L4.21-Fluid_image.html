Παράδειγμα ευέλικτων εικόνων

Επίλεξε μία από τις 3 προσεγγίσεις και μετά άλλαξε το viewport από τα DevTools.

1) Χωρίς ευέλικτη εικόνα (σταθερό πλάτος)

Η εικόνα έχει width: 600px. Κάτω από 600px viewport προκαλεί υπερχείλιση.

CSS: .fixed-img { width: 600px; }

2) Ευέλικτη με max-width: 100%

Η εικόνα δεν ξεχειλίζει ποτέ και προσαρμόζεται στο viewport.

CSS: .maxw-img { max-width: 100%; height: auto; }

3) Συνδυασμός min-width & max-width

Η εικόνα μπορεί να προκαλέσει υπερχείλιση κάτω από 350px γιατί ορίζουμε min-width:350px.

CSS: .minmax-img { width:auto; max-width:100%; min-width:350px; height:auto; }
